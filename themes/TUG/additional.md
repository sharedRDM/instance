# TUG instance installation

TUG theme holds all specific files (templates, assets, etc.) that are needed for the instance installation with this specific theme.

## Environment split

Different environments have different invenio.cfg - therefore we split up into **dev** and **qa**. Add into these subfolders any file that Dockerfile needs and has differences between environments. Add the generic ones in **base**.
